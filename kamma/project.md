# Project Guide

## What It Is And Why
This project is an ongoing app that provides a fast way to access the latest DPD database and other Pali and Sanskrit dictionaries. It exists to help people work with ancient Buddhist texts without delay.

## Who It Is For
The app is for monastics, scholars, students, and meditators interested in ancient Buddhist texts.

## One-Off Or Ongoing
This is an ongoing project that will continue to be maintained and improved over time.

## What It Will Produce
The project will produce a fast dictionary app that gives users access to the latest DPD database and other Pali and Sanskrit dictionaries.

## How You'll Know It Worked
The app is quick to open, quick to search, and updates itself and the database reliably.
